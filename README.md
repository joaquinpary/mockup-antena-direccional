# How to use 
The project uses a `config.json` file with the following structure example:

```json
{
    "satellite":{
        "sat_id": "59910",
        "latitude": "-31.435571",
        "longitude": "-64.193641",
        "altitude": "430",
        "seconds": "1",
        "api_key": "589P8Q-SDRYX8-L842ZD-5Z9",
        "delay": "60"
    },
    "network":{
        "ssid": "SSID",
        "password": "PASSWORD"
    }
}
```
The apikey is generated on the n2yo page.
